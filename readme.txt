== Description ==

SnapAI is the ultimate **free AI image generator for WordPress**. Instantly create unique, high-quality images for your website, blog posts, and featured images—right from your WordPress dashboard.

Powered by **Pollinations.ai** (no API key required), SnapAI offers unlimited AI image generation, so you can create as many custom graphics and visuals as you need, at zero cost. Streamline your workflow, supercharge your visual content, and keep your WordPress site modern and beautiful—all with just a few clicks.

= Main Benefits: =
- 100% Unlimited and Free AI Image generation
- No registration or API keys required—ever
- Save generated images directly to your WordPress Media Library
- Automatic generation of SEO-friendly alt text and titles
- Supports popular aspect ratios: 16:9 (Landscape), 1:1 (Square), 9:16 (Portrait)
- Choose stylish looks: Realistic, 3D Render, Anime, Oil Painting, Cyberpunk, and more
- Native WordPress UI and smooth workflow

== Features ==

* Unlimited AI Image Generation (Free)
* No API Keys required
* One-click "Save to Media Library"
* Automatic SEO Alt-Text and Title generation
* Support for multiple aspect ratios (16:9, 1:1, 9:16)
* Modern styles (Realistic, 3D, Anime, Cyberpunk, Oil Painting, etc.)

== Installation ==

1. Upload the `snapai` plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the ‘Plugins’ menu in WordPress.
3. Go to "Media > SnapAI Generator" and start creating!

== Frequently Asked Questions ==

**Yes!** SnapAI is powered by Pollinations.ai, a fully free and open generator. You can create as many images as you want at no cost.
